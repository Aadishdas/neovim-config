# neovim_config

<details><summary> Install neovim config</summary>

```
git clone https://github.com/Aadishx07/neovim_config.git "${XDG_CONFIG_HOME:-$HOME/.config}"/nvim
```

</details>

<details><summary> Remove config</summary>

```
rm -rf ~/.config/nvim
rm -rf ~/.local/share/nvim
```

</details>
